# editide
======================
editide is a simple prototyping editor meant to accelarate responsive app development.

### Feature ( there is only one thats quite good)
* Live update

###Credits
editide uses a number of open source projects to serve its purpose:

* [AngularJS]  
* [Ace Editor] 
* [Twitter Bootstrap] 
* [Sass] 
* [jQuery] 
* [Ruby On Rails]
